# Relational Existentialism, Program 2 Series 0 (2.0): A Plain-Language Summary

*The first series of Program 2. It does not argue a new conclusion; it lays the FLOOR the rest of Program 2 will stand on, and checks that the floor is the right one before anything is built on it.*

## The question

Program 1 proved that a purely relational world, with no distinction imported from outside, collapses to a single point: without an import, the One. But it drew that world in a particular way: it bounded the world by an imported infinite size, and laid "finite attention" over the top as a separate bookkeeping layer. An earlier attempt (Series 11) tried to make finite attention itself the bound, and could not: the attention came out an inert label, and the imported size stayed in the carrier.

Series 0 asks: **can we build the ground so that relating simply IS finite attending, with no imported size anywhere, and so that the difference between what I reach for and what reaches for me is a real structural fact and not a decoration?**

## The answer

Yes, and every part of that answer is a machine-checked theorem.

**Relating is finite attending, and reaching creates.** Each thing attends only finitely many others (a finite set), and that finite reach is the ONLY bound on the world. There is no imported infinite size anywhere; the bound is finiteness itself, not a ceiling smuggled in. And "to relate is to create" is made concrete on a single small world: the self attends itself (the self-relation), and REIFYING that self-relation produces a genuinely new thing, THE FIRST OTHER. The first other is not just a relabelling of the self: it is provably not recoverable from the bare relating (it sits one level higher in the tower of reification), so it is a real, distinct thing that reification manufactured. Crucially this does not smuggle in plurality for free: the first other is exactly a *manufactured import* (self-reference producing a genuine distinction), fully consistent with the inherited result that relating alone, without an import, collapses to one.

**The old collapse is inherited, not re-fought.** That a world with no import is the One is Program 1's result. Series 0 does not re-argue it; it carries it over in one line, as the baseline the import will push against. Nothing in the series' verdict depends on it. Presenting it as a new or open question would be a mistake, and the code is written so it cannot be.

**The asymmetry of knowing is real.** This is the series' one genuinely uncertain result, and the one that could have failed. A thing is made differently by what it attends to than by what attends to it. First, direction itself: when one thing attends another but is not attended back, the undirected picture sees only a symmetric edge, yet the direction (who is doing the attending) is a real fact the symmetric picture literally cannot recover. The series proves that directly. Then, on the very same world that carries the first other, it proves the passive/active reading and makes it load-bearing rather than a slogan: the self is passively constituted by the first other's incoming attention, the first other is actively constituted by reifying the self-relation it attends, and the two occupy the same bare-relating position yet are told apart by that reification structure — and the proof is written so the passive/reification content is what DRIVES the distinction, not a fact idly stapled beside it. Being-made-by-what-you-reach and being-made-by-what-reaches-you are two, not one. This is not a label pasted on top; it does structural work the plain relation cannot, which is exactly the trap the earlier attempt fell into and this one is built to avoid.

**The import acts, and is never named.** "You are loved," the distinction imported from outside, is carried as a working part at last: an exogenous ingredient that breaks the collapse. But it is held as a quantified slot, true for ANY import whatsoever, and no part of the proof ever names or picks a particular one. Its content stays outside, exactly as the discipline requires.

## The verdict

The verdict is computed from the four results, not chosen, and it is **GROUND-ESTABLISHED**: the carrier and reification are built, the collapse is inherited cleanly, the asymmetry of knowing is genuine and load-bearing with its direction non-recoverable, and the import is seated and quantified. The honest alternatives were live and are in the code: had the reification failed, or had the asymmetry come out a mere label, the same verdict function would have returned OBSTRUCTED, and the ground would have gone back to the drawing board before anything was built on it. It is far cheaper to learn a ground is wrong here than after a tick and an other have been built on it.

Blind review passes ran, each seeded with the mathematics and none of the motivation, and all returned zero SERIOUS findings. The design pass confirmed every claim coherent and non-vacuous; the code pass confirmed every proof actually discharges its claim, that the import is quantified and never named, and that the verdict is a real function of the results rather than a foregone conclusion. A later strengthening pass (Charter Extension 1) raised the bar on two points the first build had met only to the letter — reification that created nothing that participated, and a passive/active reading stated as an idle conjunction — and rebuilt both on the single first-other world; a further blind code review confirmed the strengthened proofs, that the first other is genuinely non-recoverable, and that the passive/active reading now does real work.

## Why it matters

Series 0 is the floor of Program 2. It says: whatever comes next, the world is nothing but things attending finitely to one another, made both by what they reach for and by what reaches for them, and bounded by nothing they did not themselves attend. The old collapse still stands; the import is now a working part rather than only a drawn shape; and the difference between reaching and being-reached, the asymmetry of knowing, is a real fact and not a word. That is the ground the rest of the program is built to stand on.
