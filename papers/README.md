## Status

Both papers are first drafts. Formal claims cite the Lean artifacts in this
repository (`program-1/series-07/formal/` for the Parmenides paper); the plan of
record is to extract the minimal Lean required into a standalone,
program-vocabulary-free artifact repository with a claims table, CI, and a
Zenodo DOI before submission.

House rules for these manuscripts:

- Machine-checked claims and interpretive glosses are kept strictly apart, as
  everywhere in this repository.
- The papers use no private program vocabulary; every term is defined in the
  paper or standard in its literature.
- No em dashes.
