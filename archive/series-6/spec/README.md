## 1. The one-paragraph design

The charter proves (WS2) that any *static* construction that is *genuinely* globally atomless — atomless at the carrier, importing no external atom (label, index, weight) to distinguish its objects — collapses to a point; every static plurality is bought by an imported atom (`νLk`'s labels, the tower's index). The repair (WS1, WS3–WS6) is to build not a finished object but a **process** over the honestly atom-free plain carrier `νPk`, whose plurality is carried by **endogenous time** — the founded history generated by the diagonal, the one distinguisher that is not an imported atom. The engine (WS3) is the identity *fixed-point-freeness = incompleteness = non-termination*: the self-survey provably cannot close (Cantor/Lawvere), and the residue it misses is the content of the next moment. The arrow (WS4) is self-knowing's one-directional lossiness. Whether the many local arrows knit into one global time or a partial causal order (WS5) is decided by whether the two-sided survey must agree, which plurality forbids. WS6 delivers the headline — genuinely atomless *and* plural — and asks whether groundlessness, plurality, and time are one engine or a conjunction; WS7 audits that.

## 2. Ambient theory (shared by all workstreams)

**Base carrier.** The **plain** κ-bounded powerset terminal coalgebra `νPk κ` (Series 4 `νPk`, transcribed), *not* the labelled `νLk`. This choice is load-bearing and is the charter §3/§5 sharpening: `νLk`'s plurality is bought by an imported label alphabet `Q` (a bare label is an atom moved into the alphabet), so building on `νLk` would forfeit *genuine* atomlessness before the process even starts. We build where the static collapse is honest.

**The functor.** `F X = P_κ X`, the κ-bounded powerset functor (Series 4 `PkObj`/`PkMap`). No labels, no `Q × (-)`. Anti-foundedness is *modelled* by the terminal coalgebra (Aczel via Rutten/Jacobs), as in Series 4; no AFA axiom.

**The finite-approximation tower.** The final ω-chain of `F`:
```lean
def Approx (κ : Cardinal.{u}) : ℕ → Type u
  | 0     => PUnit
  | (n+1) => PkObj κ (Approx κ n)          -- a < κ-bounded set of stage-n approximations
notation "A" => Approx
def trunc (κ) : (n : ℕ) → A κ (n+1) → A κ n   -- π_n, the final-chain projection
  | 0     => fun _ => PUnit.unit
  | (n+1) => fun s => PkMap (trunc κ n) s
```
`A κ 0 = PUnit` is the **closed bud** (the terminal object, "unopened"); `A κ (n+1)` is a `< κ`-bounded set of stage-`n` approximations; `trunc` forgets the deepest layer. Every `A κ n` is a **founded** object (finite descent to buds). Transcribes cleanly from the QPF machinery underlying `νPk`.

**The process carrier (fixed by WS1).** A **process** is a compatible thread of founded approximations:
```lean
def Proc (κ : Cardinal.{u}) : Type u :=
  { x : (n : ℕ) → A κ n // ∀ n, trunc κ n (x (n+1)) = x n }
```
with **stagewise identity**: `x = y ↔ ∀ n, x.1 n = y.1 n`. The world is `Proc`, and we **do not** quotient it by bisimulation / pass to the terminal-coalgebra identification — that quotient is exactly the completion the Static Collapse is a fact about. `Proc` is guarded recursion realised concretely in plain Lean (the topos-of-trees object without the topos library); the metric and pro-object encodings are WS1's pre-registered alternatives.

**Bud, atom, and dynamic atomlessness.** At stage `n`, a node whose content is `PUnit` is a **bud** — "not yet unfolded", not an atom. A **real atom** is a bud that stays a bud forever. **Genuine global atomlessness is a liveness property of the process — productivity:**
```lean
def Productive {κ} (x : Proc κ) : Prop :=
  ∀ n, ∀ b, IsBud (x.1 n) b → ∃ m, n < m ∧ Opened (x.1 m) b   -- every bud opens eventually
```
No imported atom is available to carry distinction (no label alphabet); the only distinguisher is the **founded history** `n ↦ x.1 n`, which is generated endogenously by the diagonal (§below). This is the charter's "time is the one distinguisher that is not an imported atom."

**Moments, not just states.** The fixed-point-free time-step and `≺` live on **moments** (a state at a survey-depth), not bare states. This is forced by Ω (see §4): a state can recur while its moment strictly advances. A moment is a pair `(x, n)` (a process observed through depth `n`), or equivalently a finite founded history-prefix.

## 3. The diagonal time-step `Δ` (fixed by WS3, consumed everywhere)

The **self-survey** at stage `n` is a map from stage-`n` states to their stage-`n` predicates; by the finite Cantor diagonal (Series 4 `ws6_lawvere_incomplete`, transcribed) it is **not surjective** — there is a **residue** `d_n` it misses. The **time-step**
```lean
def Δ {κ} : Proc κ → Proc κ        -- opens, at each stage, the bud carrying the residue d_n
```
is defined so that `Δ x` at stage `n+1` opens a bud with the residue of `x`'s survey at stage `n`. **`Δ` has no fixed *moment*** — no moment equals its own successor — precisely because the survey is provably non-surjective (Cantor). This realises the charter's **engine identity**:

> fixed-point-freeness (no moment at rest) = incompleteness (the survey misses `d_n`, Cantor) = non-termination (the process never halts / `Productive`).

The **residue is the successor**: WS3 must make `Δ` *literally* the residue-opening, surviving the strip of "diagonal" (charter §7 / protocol §0.4). If it cannot, WS3 reports **Trivialized**.

## 4. Moments, the order `≺`, and the endogenous face (fixed by WS4/WS5)

**The order.** A single definite relation, consumed by both WS4 (directionality) and WS5 (totality):
```lean
def prec {κ} : Moment κ → Moment κ → Prop        -- m ≺ m'  iff  m' opens a residue of a survey at m
```
i.e. the extension order on founded histories generated by `Δ`. WS4 proves `≺` is **strict and directional** (the survey/truncation `trunc` is the lossy, deterministic, many-to-one *past* map; residue-opening is the one-to-many *future*), and **endogenous** (no external ℕ, no first moment — a first moment would open a residue of nothing). WS5 proves whether `≺` is **total** (global time) or **partial** (relativity).

**The endogenous two-sided face.** Series 4 imported labels to get a face; Series 6 reads the face off the survey structure, so it is internal and costs no imported atom:
```lean
def faceAt {κ} (a b : Proc κ) (n : ℕ) : A κ n     -- the part of a's stage-n approximation engaged in surveying b
```
It is a genuine sub-object of `a`'s current founded state (analogue of Series 4 `face`/`ReachSet`, now temporal), and **two-sided**: `faceAt a b` and `faceAt b a` need not agree. This mismatch is the lever of WS5: `≺` is total iff the two-sided surveys must agree; plurality forbids agreement (a single agreeing knower is the §3 collapsed point), so `≺` is a partial causal order — unless Ω acts as a shared reference (the pre-registered absolute-frame branch).

**Ω, dynamically.** `Ω_n :=` the depth-`n` approximation of the self-loop `{Ω}` (Series 4 `omegaState` truncated). Ω is `Productive` and is the **canonical non-terminating orbit** of `Δ`: its self-survey is complete *in extent* (it faces all of itself) yet **closed at no depth** (Series 4 `ws6_omega_nonterminating`, transcribed), so the *state* Ω recurs while its *moment* strictly advances — Ω is not a rest state, it is the pure clock. This is why `Δ` is fixed-point-free on moments though Ω recurs as a state.

## 7. Predicted headline

A genuinely built, non-collapsing process over `νPk` that is **genuinely globally atomless and plural** — plurality carried by endogenous time, no imported atom (the achievement no static object can have); the **Static Collapse** as the motivating Impossibility; the **engine identity** and **endogenous arrow** earned; the **relativity verdict** most likely *partial order* (forced by the two-sided endogenous survey), with the Ω-absolute-frame branch checked; and the **one-engine unification** the signature risk — expected, per the program's pattern, to land at `payoffsEstablished` (a conjunction with genuine derivations) unless WS6 proves groundlessness and plurality the *same* residue mechanism. Sharp negatives that count as success: the Static Collapse (WS2) and, if `≺` is forced partial with no strict global-time frame, a no-absolute-time impossibility (WS5).
