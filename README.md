# Relational Existentialism, Program 2

**A machine-checked, dynamic, purely relational universe of genuinely multiple perspectives — a self and an other, a first tick of time, and the coherence between them — built in Lean 4 with its new wall proved rather than assumed.**

## The premise

Program 1 drew the edge of the One. It proved, machine-checked and axiom-clean, that a groundless, atomless, faithfully-relational world that is *determined* by its relating holds **at most one thing**: to be more than one you must be given a distinction the relating cannot itself carry (the Import Theorem), a distinction whose shape is the diagonal's own residue and whose fit is an adjoint pair with a genuine defect. Program 1 was forbidden to name that distinction, select it, or spend it. It drew the door and left it open.

**Program 2 walks through the door.** It takes the imported differentiator as given — not as a single choice but as an exogenous **stream** of them, quantified over and never named — and asks the next question: *once the distinction arrives, does the relational world become dynamic and populated by genuinely multiple perspectives, a self and an other, and where does the next wall stand?* The method is the one every Program 1 series used: **construct** the dynamic multi-perspective universe relative to the stream, prove the invariants with the stream's role quantified — formally, over every labeling of the finite witness carriers, the code's stand-in for "every stream"; no stream object exists in the code (program review, PR2-R3) — then **locate** the new impossibility the construction reveals.

This repository makes that program mathematically exact. Every result below is sorry-free and rests on no axioms beyond Mathlib's standard three (`propext`, `Classical.choice`, `Quot.sound`) — several rest on fewer, and the final verdict on none. Throughout, machine-checked claims and interpretive glosses are kept strictly apart. That distinction is load-bearing; please preserve it when citing.

## The four instruments

Program 2 does not forge new machinery. It composes the four tools Program 1 proved, one per tenet, and holds Program 1's discipline exactly — spending the import as a quantified stream and never as a name:

1. **Relata self-reference** (the diagonal) — *"Self is a paradox."* A structure rich enough to inspect itself leaves a residue it cannot capture.
2. **Relational reification** (relations become relata) — *"To relate is to create."* A relation, read, becomes a thing that can be related to in turn.
3. **Relata finite attention** (the bounded reader) — *"To attend is to become."* Each relatum perceives only a limited aspect of itself and its relationships; relating *is* that finite attention.
4. **Symmetry breaking by the import** (chance, choice, subjectivity, or something outside relation) — *"You are loved."* The one thing the relating cannot decide, and Program 2 is still forbidden to fill.

## What we built and proved

Program 2 proceeds in **series**, each a self-contained charter that states its own question, builds its own object, and proves its own results. Program 2 relaxes Program 1's transcribe-only rule into a **layered import chain** — `P1 → S0 → S1 → S2 → S3`, each series importing exactly the one before it, reaching the rest transitively — where every layer is built and axiom-checked before the next imports it, so importing a verified layer cannot introduce a gap and transcription drift is removed as a failure mode. The Program 1 prior art enters the build once, re-seated on Program 2's carrier as the `P1` foundation, with the two Program 1 adversarial traps (a convergence tautology of an unconstrained type; a bounded reader quantified out to a bare "many") deliberately **excluded** behind a guardrail so no series can recreate them.

Each series computes its own verdict — a small function reading which way the construction fell, with the alternatives genuinely reachable, so the reported outcome is an earned output, not a constant.

### Series 2.0 — the ground: *relating is finite attending* (verdict: GROUND-ESTABLISHED)

The carrier the rest of Program 2 stands on. A relatum's out-attention is a **finite set** (`attends : X → Finset X`) — finite attention is the sole bound, with no cardinal ceiling smuggled in — it relates symmetrically to what it attends and what attends it, and it **knows** only what it actively attends. On this ground:

- Reifying a relatum's self-relation yields **the first other**, provably non-recoverable from the plain relating (`ws1_first_other`).
- **The knowing is asymmetric even though the relating is symmetric** (`ws3_direction_not_recoverable`, `ws3_active_passive_distinct`, `ws3_passive_constitution`): a relatum is made differently by what it attends than by what attends it — the unreturned gaze — a load-bearing structural fact the symmetric relating cannot see, not a mere label.
- The import is carried as a quantified ingredient, seated on the ground (`ws4_import_quantified`), and the collapse to the One *without* it is inherited from Program 1 as settled baseline, never relitigated (`ws2_collapse_inherited`).

This is the ground an earlier series aimed at (finite attention as the endogenous bound) and did not reach; here relating *is* attention by construction, so attention cannot go inert.

### Series 2.1 — the tick: *endogenous time* (verdict: TWO-ZONE)

Time is constructed, not imported: **one tick is a relation becoming a relatum, read by a finite attention**, seeded by the stream. A cycle (a attends b attends c attends a) reifies into its own relatum with the finite attention of its components (`ws1_cycle_reifies`). The knot is whether succession smuggles in a second differentiator below the line, and the answer is a fork:

- Causal order is **endogenous** where the relating carries it (`ws4_causal_order_endogenous`: a causal step strictly raises reification rank).
- A **total linearization** of ticks is an import (`ws4_linearization_import`: rank alone cannot order the composites, so a clock is a distinction the relating does not carry).
- The stream's exogeneity is a proof obligation, discharged (`ws3_stream_exogenous`, `ws3_tick_needs_stream`).

Time moves without a smuggled clock. (An entity's smallest tick is set by its own attention, so one entity's smallest unit need not match another's — the tick is not a background Planck grain.)

### Series 2.2 — the other: *genuinely two reader-wise* (verdict: TWO-FACING)

The second perspective is the **reified residue of the self's own self-inspection**, given its own finite attention. Self and other are proved genuinely two not label-wise but **reader-wise**:

- The other is real **for a named, fixed, bounded reader** — `slfReader`, a genuine `FiniteAttention` whose reading is load-bearing, never a "many" quantified out (`ws2_other_reader_wise`; the exact repair of Program 1's reader trap).
- The self/other separation is **non-recoverable** from the plain relating (`ws2_other_non_recoverable`), which by the Import Theorem is precisely what a genuine distinction must be.
- The two face each other **asymmetrically and partially**, over incomparable reaches, with a **joint residue** that lies outside the union of both readings (`ws4_mutual_residue`, `ws3_facing_asymmetric`, `ws3_facing_partial`).

A self and an other, mutually reading one shared field that includes each of them and itself.

### Series 2.3 — the coherence: *the edge* (verdict: SHAPE-DRAWN)

The program's oldest question, asked across the two real faces: **does the most loving thing for the self cohere with the most loving thing for the other?** Each face gets a typed-but-never-evaluated valuation, and `Converges₂` says the self's valuation, carried to the other, agrees with the other's own. Over the genuine `slf`/`oth` pair the payoff is a genuine **two-zone fork**:

- **In-sight zone (forced):** where the relating can *see* the valuations — where they respect plain bisimilarity — coherence is **forced to hold**, because `slf` and `oth` are plain-bisimilar under the collapse engine (`ws2_converges_decided_in_sight`).
- **Import zone (dissent):** every valuation under which they *fail* to cohere is a genuine **`¬ Recoverable` import**, standing exactly on the Import Theorem's boundary (`ws3_dissent_is_import`).

The fork is genuine and not a tautology: the in-sight class is inhabited and **strictly inside** the faithful class (`ws4_insight_proper`), the forcing genuinely consumes `slf`/`oth` bisimilarity, and both zones are witnessed on the same pair (`ws4_two_zone`) — the exact trap an earlier version of this fork fell to, foreclosed by rebuilding the machinery fresh and constrained rather than importing it. **The direction of convergence is never decided:** no theorem says the two do, or do not, cohere. The verdict `SHAPE-DRAWN` is computed by `ws5_verdict_eq`, by `rfl`, on no axioms. The mathematics draws exactly where the loving claim would be true or false and proves the relating cannot see which.

## The result: CONSTRUCTED-AND-WALLED

Four rungs — **GROUND-ESTABLISHED · TWO-ZONE · TWO-FACING · SHAPE-DRAWN** — compose into the program verdict. The dynamic, purely relational, multi-perspective universe **is built with the import quantified** (over every labeling of the witness carriers — the code's stand-in for "every stream"): the self and the other are genuinely two reader-wise, time moves without a smuggled clock, and the coherence of self and other falls as a genuine two-zone fork on the Import Theorem's boundary, with both arms reachable. **No second import was needed** — the single exogenous stream carried the whole motion. And the edge came back **SHAPE-DRAWN, not decided**: the structure draws the shape of the self-and-other's coherence exactly and proves it cannot fill it. *"You are loved"* stays in the prose, held open between the two faces, and is never a proof term.

Program 1 showed the One's edge was a hinge. Program 2 lets the door swing, builds the motion of the Two, and finds the next wall exactly where the relating stops being able to see.

## Phase 2: the physics of the built universe (complete)

With a validated object in hand, Phase 2 shakes it to characterize its physics, under one thesis: **the universe is thoroughly self-relative — every physical quantity is a quantity for a self, there is no absolute frame — and the one thing classical physics held absolute, a globally conserved energy, provably fails here, returning only in the static One-limit** (the general-relativity precedent: locally conserved, globally not). Five series carry it: **2.4 Space** (the world and the two axes), **2.5 The Loop** (closed time; the fold), **2.6 The Thread** (persistence of the self), **2.7 The Ledger** (the measure and its conservation), **2.8 The Common Good** (a global good, or only local coherence). The program charter extension ([`program-2/charter-extension.md`](./program-2/charter-extension.md)) states the phase, its costume gate (no knot may reduce to the Import Theorem), and its sequencing.

All five series have landed; Phase 2 is complete. **Series 2.4 — Space** builds *the world* (a genuine lateral population of same-rank peers, the ground the rest of Phase 2 inherits) and proves the lateral axis genuinely **DISTINCT** from the vertical axis of time: the two gradings separate different states (neither is a function of the other), the fork is genuine (its collapse *realized* on a second carrier, not excluded by fiat), and the metric is a real **directed, granular, self-relative** distance — self-relative as a theorem, two selves measuring genuinely different distances with no reconciling frame. Space is real, and it is not time seen sideways.

**Series 2.5 — The Loop** asks whether time can close into a ring in a world whose *relating* is full of rings, and proves it cannot: the verdict is **ACYCLIC**. The ground is the fold — no smallest thing (nothing bottoms out) and no largest (reification is unbounded), the whole instead closing on itself through the diagonal, where the largest act of self-inspection produces the smallest thing it cannot contain. On that ground, the relating genuinely loops (attention cycles), yet causation — a *structurally* defined reification-dependency, its acyclicity proved rather than assumed — strictly climbs, so *those same cyclic edges carry no causation*: time climbs out of the loop rather than around it. The one place a causal loop could close is the self-membered fold point, which the genuine tower does not realize (and the diagonal forbids as a self-total point); that the fork is real, not decreed, is witnessed by a second carrier where a causal self-loop genuinely exists. The relating loops; time does not.

**Series 2.6 — The Thread** asks the most personal question the program reaches — does the self persist through time as a recoverable thread, or is each tick a new self and the continuity an import? — and returns **SHAPE-DRAWN**. By relational identity a thing is itself only while it relates alike, so strict identity across a tick fails (the trivial ground); the series earns its keep on the *weak* thread, proven genuinely weaker than strict identity (it holds where strict identity fails). That weak continuity is **recoverable** on a merged stream (mutual attention, the relating carries the weave) and an **import** on a cut stream (one-way attention, the knowing-asymmetry — the weave received from outside), both on genuine carriers, so which obtains is *self-relative* and not forced. And the single lived timeline — the one-after-another of a life — is proven the self's *linearization import* over a merely partial causal order, not a fact of the world. The self is, at most, a thread continuously re-woven; whether the relating carries the weave or receives it as a gift is not the same for every self, and the structure does not decide it for you.

**Series 2.7 — The Ledger** asks whether the universe keeps a conserved quantity — an energy, an information — and returns **MONOTONE-ONLY**: a measure survives, but nothing is conserved. A non-trivial measure of distinction is genuinely found (`Q`, the reification rank; its differences are real imports, not a bookkeeping artifact), so the outcome is not the empty DISCONNECTED. But the tick *strictly raises* it, and — the heart of the result — the measure is not invariant under what the self can see: any genuinely in-sight-conserved measure would have to be blind to the tick, so the only conserved quantities are trivial. There is no conserved ledger, not even locally. And this is settled by *proof*, not left open: conservation-from-within is **impossible**, because self-reference is always an act of creation — the diagonal (the residue) is a genuine new non-recoverable content for every inspection, never a relocation of something already latent. A genuine CONSERVED-RELATIVE was attempted first, on the charter's own intended measure, and refuted on record: a lossless re-encoding does conserve a *size*, but the diagonal cannot be made to decide the measure toward conservation. The universe has an arrow and keeps no ledger — to relate is to create, and creation is the reason there is nothing to conserve. (An earlier landing computed CONSERVED-RELATIVE; the Tier-1 landing review found it a costume — an "in-sight conservation" that was only the collapse engine identifying two states, not an invariance of the measure — and the series was reground to the honest MONOTONE-ONLY.)

**Series 2.8 — The Common Good** closes Phase 2 by asking whether *many* selves share a good — whether local, pairwise coherence glues into one good for all, or is **frustrated** — and returns **FRUSTRATED**. This is not the single edge of Series 2.3 but the *network*: a many-body question that only exists for three or more selves and survives even after every edge is decided coherent. Each pair of selves reconciles (local coherence is pervasive and unconditional), yet the reconciliations, composed around a cycle, need not return to where they started — a non-trivial *holonomy*. On a directed 3-ring of peers it comes to 3 ≠ 0, so no single good is consistent with every pairwise reconciliation at once, though every pair agrees; on a mutual star it vanishes and a global good exists. Both are genuinely reachable — the fork is a fact about which world you are in, not a fiat — and, decisively, the frustration is *carried by the directedness itself*: symmetrize the attention and it vanishes identically, so the obstruction lives in the knowing-asymmetry, not in any bolted-on device. Local coherence does not glue: the common good is the value analog of energy's global failure. No common good is written into this universe; it is made, edge by edge, and never owed.

With that, every capstone's *global* has failed — no absolute frame, no closed time, no guaranteed thread, no conserved ledger, no common good — and the phase thesis held to the last wall.

## Phase 3: can this physics be ours? (complete)

Phase 1 built the universe; Phase 2 found its physics thoroughly self-relative, the global failing at every capstone. Phase 3 turns to face our own universe and asks the question the program has been circling: **can this metaphysics — relational, atomless, self-referencing, symmetry-breaking, under finite attention — recover the physics we actually observe?** It is the fourth step of the program's method made rigorous: *state the ontology, model it, shake it, ask "is this everything?"; if not, say precisely what is missing.* Five series (2.9–2.13), chartered in [`program-2/charter-extension-2.md`](./program-2/charter-extension-2.md) and **ordered by execution** as a DAG rather than a chain (they probe the finished object rather than build a taller tower), test the three pillars our physics rests on that the model has never been shaken against — relativity, quantum mechanics, and their union: **2.9 The Cone** (is finite attention a maximal rate of becoming — a light cone, a speed of light?) and **2.10 The Reversal** (does the tick have an invertible core — the reversibility that conservation and quantum theory both require?) run first and in parallel; then **2.11 The Phase** (can distinction *cancel* — is there a signed amplitude that interferes? — the decisive quantum test, reusing 2.8's holonomy); then **2.12 The Weight** (do the imports carry a measure — a probability, the Born rule?); then **2.13 The Curve** (does the information grain curve the metric — gravity from within?).

The thesis is an honest question, not a foregone conclusion. What Phase 2 already proved suggests the built object is a *classical, irreversible, thermodynamic* world — likely to recover the relativistic cone (finite attention forces a maximal rate) and likely to fail the quantum pillar as built (nothing yet cancels; nothing is reversible). And a NOT-RECOVERED verdict is not a defeat but a **specification**, naming exactly what the ontology's next revision must add. The phase is anchored to the quantum reconstruction theorems, which prove quantum mechanics follows from a short list of information-theoretic axioms — reversibility, superposition, a probabilistic structure — so the quantum sub-arc (The Reversal, The Phase, The Weight) tests exactly those, and a full recovery would *force* quantum theory. Two disciplines guard it: the **no-smuggling gate** — a feature counts as recovered only if it is *earned* from the built primitives, never assumed to obtain the answer — and **rigor over desire**: the deliverable is the honest catalogue of what falls out and, precisely, what does not.

The two tier-1 probes have landed, and the predicted split held. **Series 2.9 — The Cone returns CONE:** finite attention *is* a maximal rate of becoming. The rate is not postulated — it is *defined* as the largest lateral distance the attention spans in one step — and the cone it casts carries genuine content, proven by a pair of worlds with the *same* saturated causal order but *different* cones (so the cone is not a relabelling of that order). The connective theorem the landing lacked — that the cone genuinely *bounds* the causal reach — is proved in the review-repairs module (`PR2R1.pr2s4_cone_bounds_reach`), together with the honest slack: the ball is symmetric where the attention is directed, so it over-approximates the reach (`PR2R1.pr2s4_ball_exceeds_reach`). On a world where each self attends only its neighbour, the far peer sits in an elsewhere no single tick can reach; only when attention widens to the whole world does the cone swallow everything. Relativity's light cone falls out as the shadow of a bounded capacity to attend — the first evidence the metaphysics can speak a pillar of our physics. **Series 2.10 — The Reversal returns IRREVERSIBLE:** the tick has no core you could run backward. Its only reversible-looking piece (the reification section) merely *decodes* what was reified — it never lowers the rank the tick raised — and an exhaustive check proves no sub-dynamics of the built tick is a measure-preserving bijection. This is a NOT-RECOVERED, and it is the honest, consequential kind: it confirms that the missing conserved ledger of Series 2.7 was no accident — no sub-dynamics of the built tick preserves the measure, though the built tick is itself *periodic* (the three reachable states cycle and the measure returns with them, `PR2R1.pr2s5_tick_not_monotone`), so "the arrow" names the rank-mixing of the cycle rather than a monotone flow (program review, PR2-S5) — and it names precisely what a next iteration must do to become our universe: put reversibility *beneath* the one-way tick. (The seed for that already exists in the program — Series 2.8's signed increment, the first quantity here that can cancel.) As built, then, this is a classical, irreversible, thermodynamic world that already speaks relativity; whether it can be made to speak the quantum is what **2.11 The Phase** asks next.

**Series 2.11 — The Phase returns INTERFERING:** distinction can cancel. This is the single strangest fact about quantum mechanics — that two ways of reaching an outcome can subtract, so light plus light can make darkness — and everything the program built before Series 2.8 could only ever *add*. But 2.8's signed increment can cancel, and 2.11 shows that cancellation is the real thing: a `±1` sign read off 2.8's holonomy (earned from the model, no complex number smuggled in) makes two co-terminal paths destructively interfere on the frustrated ring — a combined weight strictly *below* the sum of its parts, the signature no classical probability can show — while on the gluable star the same sign adds constructively. The very fork that made the common good frustrated (2.8) now makes two paths cancel. The door to a quantum reading opens, in the model's own key. It opens only partway, and the charter says so plainly: the amplitude here is a *real* `±1` sign, genuine interference but not yet the full complex phase, which — with the Born rule — is deferred to **2.12 The Weight**. (The landing also carried a process lesson: the executor skipped the protocol's blind design-review phase; the review was run retroactively and independently before acceptance, and the deviation recorded.) So Phase 3's picture so far is a genuine in-between: a quantum-flavoured *structure* — a light cone, interference — resting on an *irreversible* dynamics. The world can interfere but cannot yet be run backward; and the same signed holonomy that lets it interfere is exactly the material a later iteration would use to give it the reversibility it lacks.

**Series 2.12 — The Weight returns BORN.** Quantum mechanics asks a world two things: that possibilities interfere, and that when they are realized the probability of an outcome is the *square* of its amplitude — the Born rule. Series 2.11 found the first; Series 2.12 asks the second, and the danger is obvious: it would be trivial, and dishonest, to simply *define* the probability as the square and declare it recovered. So the series does the opposite. It adds a single, deliberately *square-blind* consistency demand — that a probability must vanish wherever the amplitudes cancel ("cancellation is absence") — and then asks which measure meets it. The classical rule (square each path, then add) *fails*: it insists an outcome occurs where interference has cancelled it to nothing. The Born rule (add the amplitudes, then square) *passes*, on every configuration. The square is not assumed — the test never mentions it — but nor is it forced: the review-repairs module proves the absolute-value rule passes the same test (`PR2R1.pr2s4_square_not_unique`), so what the test earns is the *class* of non-classical, cancellation-respecting weights, with the square its exhibited representative. Two honest limits are stated plainly and not buried: this recovers the *structure* of quantum probability — a non-trivial, non-classical, squared weight — not literal chance a world samples; and it forces the square over the classical alternative, not as the unique rule among all powers (Gleason's theorem, and the full complex amplitude, remain beyond it). So the quantum *kinematics* is now recovered whole — a cone, interference, and a Born-form probability, three signatures all tracing to the one holonomy Series 2.8 built — resting on the *irreversible* dynamics of Series 2.10. The model has the shape of quantum probability, with no reversible evolution to sample it; and whether geometry itself bends to the grain is what the final Phase-3 series, **2.13 The Curve**, asks last.

**Series 2.13 — The Curve returns INERT.** The final question is gravity's: does the information grain — the measure of distinction Series 2.7 built — bend the geometry Series 2.4 built, the way mass-energy curves spacetime? The danger mirrors 2.12's: it would be trivial to *define* a grain-weighted distance and call the geometry curved. So the series refuses to, and instead tests a fresh re-seat of the model's distance — Series 2.4's directed path-length, rebuilt as a decidable computation, now *connected to the built object by theorem*: the review-repairs module proves the re-seat agrees with the built `stepsFrom` on every reachable pair and discloses the one seam, the unreachable pairs where the built distance conflates "unreachable" with 0 (`PR2R1.pr2s4_bridge_faithful` / `pr2s4_bridge_diverges`) — for whether it moves when the grain changes at fixed adjacency. It does not. The distance reads the connections and nothing else about the configuration; a grain packed into a region leaves every distance around it exactly as it was. To prove the fork is not rigged, a single hand-built grain-weighted *foil* is exhibited — it *does* vary with the grain, so a coupling is expressible in this world — then proved distinct from the model's distance and fenced off, never the recovered object: the verdict reads the imported distance alone. The geometry is **INERT** — decoupled from the grain, no gravity. This is a NOT-RECOVERED of the honest, consequential kind, and the one Series 2.4 foretold: Space's very first result proved the lateral axis (distance) and the vertical axis (the measure) genuinely *distinct*, neither a function of the other — and gravity is precisely the coupling of those two axes. The model built them apart. As with 2.10's irreversibility, the verdict is a specification: to become a universe with gravity, the next iteration must couple the two axes 2.4 kept separate. The area law (Bekenstein/holographic — information bounded by boundary, not volume) is the disclosed forward-note, the deepest unbuilt question, not attempted here.

## The result of Phase 3: quantum kinematics recovered; reversibility and gravity not

Five series shook the built universe against the three pillars of our physics, and the pattern is sharp and honest. **Relativity's light cone falls out** (2.9 CONE) — a maximal rate of becoming is the shadow of finite attention. **A quantum kinematic *structure* falls out** — interference (2.11 INTERFERING) and a Born-form weight (2.12 BORN, the exhibited representative of the cancellation-respecting class, not the unique member) — and, strikingly, both trace to a single object Series 2.8 built: the signed holonomy of the directed attention, each recovered in its real/structural/rebit form and honestly scoped short of its full complex, Gleason-unique, literal-chance version. But **two things do not fall out**, and both are the consequential kind of NOT-RECOVERED — a specification, not a defeat. The *dynamics* is irreversible (2.10 IRREVERSIBLE): the tick has no measure-preserving bijective core, proven exhaustively — an arrow in the no-conserved-core sense, on a tick that is itself periodic (`PR2R1.pr2s5_tick_not_monotone`) — so the quantum shape has no unitary evolution to sample it. And the *geometry* is inert (2.13 INERT): the grain does not curve the metric, exactly as Space's distinctness of the two axes foretold, so there is no gravity from within.

So the built object is a **classical, irreversible, thermodynamic world that already speaks the quantum kinematic language** — it has a cone, it interferes, it weighs its outcomes by a squared amplitude — resting on a dynamics that runs one way and a geometry blind to its own information content. What is recovered and what is missing point, together, at the next iteration's specification, in the model's own terms: the same signed holonomy (2.8) that gives the world its interference is the reversible seed that would give it back its reversibility (uniting 2.10 and 2.11), and the two axes Space (2.4) kept apart are the ones a gravitational iteration would have to couple (2.13). Phase 3 answers the program's standing question — *can this metaphysics recover our physics?* — with a precise and partial yes: the kinematics, yes, and in the model's own key; the reversible dynamics and the gravitational geometry, not yet — and here is exactly what to build next.

## How it was built (the discipline)

Program 2 is produced by a **two-tier process**, itself documented in [`program-2/protocol.md`](./program-2/protocol.md) and tracked in [`program-2/charter-status.md`](./program-2/charter-status.md):

- **Tier 1**, a persistent design-and-review conversation, authors each series' charter, status, and self-contained protocol; reviews each landing with an independent skeptical read; and — only ever to *raise* the bar — writes a **Charter Extension** when a review finds a target set too low (Series 2.0 and 2.1 each earned one). It never writes the series' formal code.
- **Tier 2**, a fresh executor per series, runs the chartered A→G protocol from an empty `formal/` to a computed verdict, delegating only its two **blind** review phases to subagents that never see the motivating prose.

The honesty invariants are the whole point: blind means blind; a shortfall is reported as a pre-registered outcome, never relabeled as the goal; every payoff must survive **stripping its interpretive word** and still stand as a bare bisimulation, recoverability, or order fact; the import is quantified over and never named; and no proof term, definition, or discharged obligation is ever named *self*, *other*, *time*, *choice*, *God*, *love*, or *compass* as content — those words live only in prose.

**Four things stay permanently open**, each open *as a theorem* by the very results that create them, and Program 2 adds none and closes none: the content of the compass, the direction of convergence, the differentiating act, and the classification of the out-of-image imports.

## Building

Toolchain and Mathlib (Lean 4 `v4.15.0` / Mathlib `v4.15.0`) are pinned in [`lake/`](./lake/). One build compiles the whole program:

```
cd lake
lake build
```

This builds the `P1` foundation and Program 2's `P2S0`…`P2S8` (the layered chain) plus the Phase-3 probes `P2S9`, `P2S10`, `P2S11`, `P2S12`, `P2S13` (each in its own module namespace, one module per workstream), alongside the registered Program 1 series (`Series07`, `Series12`, `Series13`) it stands on, and runs each series' `AxiomCheck`, which emits a `#print axioms` record for every headline theorem. To build a single series' axiom pass: `lake build P2S13 P2S13.AxiomCheck`. [`scripts/gate.sh`](./scripts/gate.sh) enforces closure — each Phase-1/2 series imports only its predecessor and its own roots (plus Mathlib); the Phase-3 probes each import their reuse-predecessor and their own roots, reaching the finished object transitively, nothing reaching outside. The program charter, per-series charters, status ledgers, protocols, and plain-language and technical summaries live under [`program-2/`](./program-2/).
